# Selected Labs

This folder contains selected labs from the VIVA course.
Early introductory labs are intentionally omitted to keep the repository
focused on more substantial embedded systems concepts.

## Lab 7 – UART, ADC, Internal Temperature Sensor

PIC16F1829 @ 4 MHz  
Reads internal temperature and potentiometer via ADC and outputs data over UART.

## Lab 8 – I2C LCD

PIC16F1829 @ 4 MHz  
Demonstrates I2C communication with an LCD using a PCF8574 backpack.
